Angular Task
Create an angular app to render the data coming from a backend endpoint in a tree view with search.
  • Make a tree view that renders the data provided from the Backend
    • make each level of the tree collapsible
    • {{ item.name }}
  • Enable search for the view
    • the search should be case insensitive
    • the result should still be a tree view showing the whole path to the found result
  • You also should use the following:
    • tailwindCss to style components
      • This project uses only 3 css code style properties

      background: linear-gradient(45deg, rgb(176, 255, 2) 0%, rgb(255, 0, 90) 100%);

      height: 100vh;

      font-family: ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Helvetica Neue, Arial, Noto Sans, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", Segoe UI Symbol, "Noto Color Emoji";

    • state management (NgRx - component store)
      • All array actions go through ngRx
    • angular reactive forms
      • Search works through the form
